## 📂 Getting Started

To run this project locally:

```bash
# Clone the repo
git clone https://github.com/OmarElflah02/Ai-Revolution-template.git

# Go into the project folder
cd Ai-Revolution-template

# Install dependencies
npm install

# Run the project
npm run dev
```
